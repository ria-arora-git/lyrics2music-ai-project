# lyrics2music-ai-project

An emotion-aware AI system that analyzes song lyrics and generates
music based on inferred emotional intent, energy, and structure.

Instead of direct black-box generation, this project explicitly models
how lyrics translate into musical decisions.

## Key Features
- Emotion detection using Transformer models
- Energy & emotional arc analysis
- Interpretable music decision layer
- Local procedural audio generation (no cloud dependency)

## Pipeline
Lyrics → Emotion → Energy → Structure → Musical Intent → Audio

## Run
python main.py
